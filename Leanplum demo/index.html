Leanplum Plugin Demo

The Leanplum plugin gives your app access to the powerful Leanplum SDK. Optimize mobile content, send messages to (segments of) your users, analyze app usage, and more!

Custom plugins don't work in the simulator and the AppBuilder companion app, so inside Telerik AppBuilder choose Run - Build - iOS/Android - App package.

Track Events

Leanplum tracks basic usage of your automatically, but you track custom events as well. After clicking these buttons go to your Leanplum analytics dashboard and filter on these.

Push Notifications

Leanplum can send all kinds of messages to your app. One of those is Push Notifications - read about how to set those up on the Leanplum website.
